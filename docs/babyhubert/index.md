# BabyHuBERT

BabyHuBERT is the underlying encoder used as base for training VTC2.0.

## What BabyHuBERT is *not*

- **Not a voice type classifier** — that is [VTC2.0](../index.md). BabyHuBERT gives you representations 

You cannot use BabyHuBERT as VTC
